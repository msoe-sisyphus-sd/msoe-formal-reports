# What is this repository for?

This is the repository for all technical documents pertaining to the MSOE Senior Design team's 2020-2021 project work with Sisyphus Industries.  This repository contains the following:

* Final Report - this document summarizes the work of the team from the year. It is a summary for the project and is not comprehensive.

* Team Components Report - this document is a more detailed document demonstrating how the team has met ABET requirements for a software engineering project.

* Final Architecture Diagram - this document is located in both the Final Report and the Team Components report, but is placed here for easy access.
